# لعبة المشنوق (Hangman Game)

![Hangman Game Screenshot](https://via.placeholder.com/800x400?text=Hangman+Game+Screenshot)

## نظرة عامة

لعبة المشنوق هي لعبة تخمين كلمات كلاسيكية تم تطويرها باستخدام HTML، CSS، و JavaScript. تتميز اللعبة بدعم كامل للغتين العربية والإنجليزية، نظام نقاط وإنجازات، مؤثرات صوتية، تلميحات قابلة للشراء، ومظاهر متعددة لتجربة لعب ممتعة وتفاعلية.

## الميزات

*   **دعم متعدد اللغات**: العب باللغتين العربية والإنجليزية مع إمكانية التبديل السريع.
*   **نظام النقاط والإنجازات**: اكسب النقاط وافتح الإنجازات الخاصة بك بناءً على أدائك في اللعبة.
*   **مؤثرات صوتية**: استمتع بتجربة صوتية محسّنة مع مؤثرات صوتية للنقر، النجاح، الفشل، الفوز، والخسارة.
*   **تلميحات قابلة للشراء**: استخدم نقاطك لشراء تلميحات تساعدك في تخمين الكلمات الصعبة.
*   **مظاهر متعددة**: اختر بين المظهر الفاتح والداكن لتناسب تفضيلاتك البصرية.
*   **تصميم متجاوب**: استمتع باللعبة على مختلف الأجهزة، من الحواسيب المكتبية إلى الهواتف الذكية.
*   **هيكل مشروع منظم**: تم تقسيم المشروع إلى ملفات منفصلة لسهولة الصيانة والتطوير.

## التقنيات المستخدمة

*   **HTML5**: لبناء هيكل الصفحة.
*   **CSS3**: لتصميم وتنسيق الواجهة، بما في ذلك دعم الثيمات (الفاتح والداكن) ودعم RTL للغة العربية.
*   **JavaScript (ES6+)**: لتطبيق منطق اللعبة، نظام الترجمة، نظام الصوت، نظام الإنجازات، وتحديث واجهة المستخدم ديناميكياً.

## هيكل المشروع

```
hangman-game/
├── index.html          ← الصفحة الرئيسية للعبة
├── css/
│   └── style.css       ← كل الستايلات + الثيمات (فاتح/داكن)
├── js/
│   ├── i18n.js         ← نظام الترجمة (عربي/إنجليزي)
│   ├── audio.js        ← محرك الصوت (Web Audio API)
│   ├── achievements.js ← الإنجازات + النقاط + التلميحات
│   ├── game.js         ← منطق اللعبة الأساسي (اختيار الكلمة، التخمين، التحقق)
│   └── ui.js           ← كل ما يظهر على الشاشة (تحديث الواجهة، معالجة الأحداث)
├── data/
│   ├── words-ar.json   ← قائمة الكلمات العربية
│   └── words-en.json   ← قائمة الكلمات الإنجليزية
└── README.md           ← وثائق المشروع
```

## كيفية اللعب

1.  افتح ملف `index.html` في متصفح الويب الخاص بك.
2.  اختر اللغة المفضلة لديك (العربية أو الإنجليزية) من شريط التنقل.
3.  ابدأ لعبة جديدة.
4.  حاول تخمين الكلمة عن طريق النقر على الحروف في لوحة المفاتيح الافتراضية أو استخدام لوحة المفاتيح الفعلية.
5.  لديك 6 محاولات خاطئة قبل أن تخسر اللعبة.
6.  يمكنك استخدام التلميحات المتاحة لديك أو شراء المزيد باستخدام نقاطك.
7.  حاول تحقيق أعلى النقاط وفتح جميع الإنجازات!

## التثبيت والتشغيل محلياً

لتشغيل اللعبة على جهازك المحلي، اتبع الخطوات التالية:

1.  **استنساخ المستودع (Clone the repository)**:
    ```bash
    git clone https://github.com/your-username/hangman-game.git
    cd hangman-game
    ```
    (ملاحظة: سيتم توفير رابط المستودع الفعلي بعد الرفع على GitHub).

2.  **فتح اللعبة**: ببساطة افتح ملف `index.html` في متصفح الويب المفضل لديك. لا تحتاج إلى خادم ويب لتشغيل هذه اللعبة.

## التخصيص

*   **إضافة كلمات جديدة**: يمكنك تعديل ملفات `data/words-ar.json` و `data/words-en.json` لإضافة أو إزالة الكلمات.
*   **تعديل الثيمات**: يمكن تعديل الألوان والمتغيرات في ملف `css/style.css` لتغيير مظهر اللعبة.
*   **تعديل الإنجازات**: يمكن تعديل شروط الإنجازات في ملف `js/achievements.js`.
